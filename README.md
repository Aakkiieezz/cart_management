# Cart management using SpringBoot + Hibernate + Thymeleaf

Used :
- Layered architecture (Controller <--> Service <--> Repository)
- Mysql Workbench
- HTML5 (Bootstrap 4)
- CRUD Functionality
- Single Page Application
